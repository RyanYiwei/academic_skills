# Academic Skills

A collection of skills for LLMs (e.g., Claude) to assist with academic research workflows.

## Structure

```
skills/
├── paper-reading/        # Reading and analyzing academic papers
docs/                     # Documentation and usage guides
```

## Usage

Each skill lives in its own subdirectory under `skills/`. Refer to individual skill READMEs for usage instructions.
